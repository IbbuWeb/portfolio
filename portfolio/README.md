# Personal Portfolio Website

This repository contains the complete source code for a modern, responsive personal portfolio website.

## Features

- **Minimalist Design:** Clean, black-and-white theme with a focus on typography and content.
- **Responsive Layout:** Desktop-first design that adapts seamlessly to tablets and mobile devices.
- **Interactive UI:** Smooth scrolling, active navigation highlighting, and subtle hover animations.
- **Zero Dependencies:** Built with plain HTML, CSS, and vanilla JavaScript. No frameworks or external libraries needed (except for web fonts and icons loaded via CDN).

## How to Use

1.  **Clone or download** this repository.
2.  Open the `index.html` file in your web browser to view the website locally.
3.  To customize the content, edit the `index.html` file directly. You can change the text, project details, and links.
4.  To change the styling, modify the `style.css` file.
5.  The `script.js` file contains the logic for smooth scrolling and navigation highlighting.

## Deployment

Since this is a static website, you can deploy it easily on any static hosting provider like:

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages
